- [Material Design Icons](https://pictogrammers.com/library/mdi/).
- [Material Design Icons Light](https://pictogrammers.com/library/mdil/).

## How to use

The code is pretty simple:

    view : Model -> Html msg
    view =
        Mdi.Icons.Normal.abcTesting
        |> Mdi.Icons
